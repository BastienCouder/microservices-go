## Prérequis

- Docker
- Docker Compose

## Lancer la stack

```bash
docker compose up --build
```

## URL utiles

- API Gateway: `http://localhost:8080`
- Frontend web: `http://localhost:3000`
- Documentation: `http://localhost:3001`
