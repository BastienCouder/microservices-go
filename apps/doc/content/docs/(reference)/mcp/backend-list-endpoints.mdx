<ReferenceShell>
  <ReferenceMain>
    <OperationHeader
      tag="MCP Tool"
      title="backend_list_endpoints"
      summary="Return the public endpoint catalog currently exposed by the MCP server."
      method="TOOL"
      path="backend_list_endpoints"
    />

    ## Authorizations

    This tool is read-only and does not require arguments.

    ## Arguments

    No arguments are required.

    ## Behavior

    The tool returns a compact route list that currently covers gateway health plus selected auth,
    user, organization, permission, billing and notification endpoints.

    ## Response

    | Field | Type | Description |
    | --- | --- | --- |
    | `base_url` | `string` | Default base URL embedded by the MCP server implementation. |
    | `endpoints` | `Endpoint[]` | Array of endpoint objects with `service`, `method` and `path`. |

    ## Notes

    Use this tool for quick agent-side discovery. For complete business behavior, the HTTP API
    reference pages in this docs portal remain the source of truth.
  </ReferenceMain>
