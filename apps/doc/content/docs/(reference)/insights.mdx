title: Insights
description: Analysis runs, AI execution, alerts and attribution reporting.

<NoteBox title="Envelope" tone="primary">
Insight-related routes use the `{ success, data }` envelope. They are designed to follow the
workspace lifecycle after projects, prompts and models have been configured.
</NoteBox>

## Analysis runs

<EndpointTable>
  <EndpointRow method="POST" path="/analysis/projects/{projectId}/analyze" auth="Session" scope="Organization" response="Success envelope">
    Starts an analysis run. Send `promptIds` or `promptTexts`, plus `modelIds` and `runType`.
  </EndpointRow>

  <EndpointRow method="GET" path="/analysis/projects/{projectId}/runs" auth="Session" scope="Organization" response="Success envelope">
    Lists analysis runs for a project. Query param: `limit`.
  </EndpointRow>

  <EndpointRow method="GET" path="/analysis/runs/{runId}" auth="Session" scope="Organization" response="Success envelope">
    Reads one analysis run with its details.
  </EndpointRow>

  <EndpointRow method="POST" path="/analysis/runs/{runId}/responses" auth="Session" scope="Organization" response="Success envelope">
    Records a model response inside a run. Body fields include `promptRunId`, `modelId`,
    `rawResponse`, `brandMentioned`, `brandPosition`, `citationFound`, `citedUrls`, `sentiment`.
  </EndpointRow>
</EndpointTable>

## Dashboards and alerts

<EndpointTable>
  <EndpointRow method="GET" path="/analysis/projects/{projectId}/dashboard" auth="Session" scope="Organization" response="Success envelope">
    Returns the dashboard payload used to summarize visibility and changes.
  </EndpointRow>

  <EndpointRow method="GET" path="/analysis/projects/{projectId}/perception" auth="Session" scope="Organization" response="Success envelope">
    Returns the latest perception view for the project.
  </EndpointRow>

  <EndpointRow method="GET" path="/analysis/projects/{projectId}/alerts" auth="Session" scope="Organization" response="Success envelope">
    Lists alerts for a project. Query param: `unreadOnly=true`.
  </EndpointRow>

  <EndpointRow method="POST" path="/analysis/projects/{projectId}/alerts" auth="Session" scope="Organization" response="Success envelope">
    Creates an alert. Body fields: `alertType`, `severity`, `title`, `description`.
  </EndpointRow>

  <EndpointRow method="POST" path="/analysis/projects/{projectId}/alerts/read-all" auth="Session" scope="Organization" response="Success envelope">
    Marks every alert for the project as read.
  </EndpointRow>

  <EndpointRow method="PATCH" path="/analysis/alerts/{alertId}/read" auth="Session" scope="Organization" response="Success envelope">
    Marks one alert as read.
  </EndpointRow>

  <EndpointRow method="DELETE" path="/analysis/alerts/{alertId}" auth="Session" scope="Organization" response="Success envelope">
    Deletes one alert.
  </EndpointRow>
</EndpointTable>

## AI execution primitives

<EndpointTable>
  <EndpointRow method="POST" path="/ai/execute" auth="Session" response="Success envelope">
    Executes a prompt against a model. Body fields: `promptId`, `promptText`, `modelId`,
    `brandName`, `competitors`, `mockResponse`.
  </EndpointRow>

  <EndpointRow method="POST" path="/ai/extract-brand" auth="Session" response="Success envelope">
    Extracts brand information from a website. Body fields: `projectId`, `websiteUrl`.
  </EndpointRow>
</EndpointTable>

## Attribution

<EndpointTable>
  <EndpointRow method="POST" path="/attribution/projects/{projectId}/events" auth="Session" response="Success envelope">
    Records one attribution event. Body fields: `stage`, `source`, `count`, `revenueCents`,
    `occurredAt`.
  </EndpointRow>

  <EndpointRow method="GET" path="/attribution/projects/{projectId}/events" auth="Session" response="Success envelope">
    Lists recorded events for a project. Query params: `from`, `to`, `limit`.
  </EndpointRow>

  <EndpointRow method="GET" path="/attribution/projects/{projectId}/funnel" auth="Session" response="Success envelope">
    Returns the funnel summary for a project. Query params: `from`, `to`.
  </EndpointRow>
</EndpointTable>

```bash
curl https://api.example.com/analysis/projects/project_123/analyze \
  --request POST \
  --header "Content-Type: application/json" \
  --header "X-Session-Token: <session-token>" \
  --header "X-Organization-ID: 42" \
  --data '{
    "promptIds": ["prompt_1", "prompt_2"],
    "modelIds": ["gpt-4.1", "claude-3.7-sonnet"],
    "runType": "scheduled"
  }'
```
